About MiniTweet

A minimal, open-source microblogging platform built with Django & Tailwind CSS — crafted to learn, experiment, and grow.

What is MiniTweet?

MiniTweet is a lightweight social media application that lets users share short messages with the world, view posts from others, and enjoy a clean, distraction-free feed experience — much like the early days of Twitter, but simpler.

It was built as a hands-on project to master core Django concepts: authentication, URL routing, template inheritance, and full CRUD operations — all wrapped in a production-quality UI powered by Tailwind CSS.

The Mission

To prove that clean, professional UI doesn't require a massive team or budget — just thoughtful design, consistent systems, and a love for the craft. MiniTweet is an example of what's possible with Django + Tailwind in the right hands.

Built by a Developer in Progress

This project is built by someone actively learning backend development, REST APIs, and scalable web architecture. Every page, every component, every line of code is a step forward — and MiniTweet documents that journey in the best possible way.
